# 🌍 CO Predictor Dashboard

Интерактивный дашборд для анализа качества воздуха и прогнозирования концентрации угарного газа (CO) на основе данных химических сенсоров.

## 📌 О проекте
Дашборд позволяет получить предсказание на основе инфорамации химических датчиков, временных рядов и метрологических характеристиках.

## 🖥 Структура приложения
Дашборд разделен на 4 ключевых раздела:
1. **👨‍💻 Информация о разработчике** — данные об авторе и целях исследования.
2. **📊 Исследование набора данных (EDA)** — обзор данных, проверка пропусков и статистические показатели.
3. **📈 Визуализации данных** — интерактивные графики распределения CO и корреляционные матрицы.
4. **🔮 Получение предсказаний моделей** — функционал для ввода параметров (температура, влажность и др.) и получения точного прогноза CO.
